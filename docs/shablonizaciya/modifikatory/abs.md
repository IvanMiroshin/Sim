# abs

Возвращает модуль от числа

## **Синтаксис**

```text
integer $var:abs;
```

\*\*\*\*

## **Пример использования**

Входные данные:

```php
Array
(
    ['myvar'] => -3.14
)
```

Шаблон:

```markup
<p data-sim="content($data.myvar:abs);"> … </p>
```

Результат выполнения:

```markup
<p>3.14</p>
```
